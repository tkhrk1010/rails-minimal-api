# Rails Minimal API
## How to create this repository
```
$ rails new rails-minimal-api --minimal --api -T
```
## How to use
First, git clone, and bundle install.  
```
$ bundle install
```  
Secondly, start rails.  
```
$ rails new
```
Then, you can open app on your browser or curl.
```
$ curl http://localhost:3000  
```
